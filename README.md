# cordova-plugin-enabledevtools

## Description

Enables dev tools on osx platform for all users of your app (right click, inspect).

## Installation

  $ cordova plugin add https://github.com/happinov/cordova-plugin-enabledevtools

## Version History

### 1.0.0
  * Initial release.
  * osx platform only.
